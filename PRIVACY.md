# Privacy Policy for Bitbucket Blue Icon

**Last Updated:** December 7, 2025

## Overview

Bitbucket Blue Icon is a browser extension that replaces the Bitbucket favicon with the classic blue icon. This privacy policy explains our data practices.

## Data Collection

**We do not collect, store, or transmit any user data.**

Specifically:
- No personal information is collected
- No browsing history is recorded
- No analytics or tracking is implemented
- No cookies are used
- No data is sent to external servers

## Permissions

The extension requires the following permission:

- **Access to bitbucket.org**: Required to modify the favicon on Bitbucket pages. This permission allows the extension to run a content script that removes existing favicon links and adds the blue icon.

## Local Operation

The extension operates entirely locally in your browser:
- All code runs on your device
- No external network requests are made
- No data leaves your browser

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted in this document with an updated "Last Updated" date.
